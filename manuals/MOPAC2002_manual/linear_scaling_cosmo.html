Linear Scaling COSMO

The COSMO solvation model has been re-written to allow large systems to be more easily calculated.  This work was done by Dr Andrey Bliznuk, Australian National University, Canberra, Australia. 

The main additions to the SCF that need to be done on each iteration for COSMO calculations are:

  • solve the system of linear equations (1) to find induced charges q on the molecular surface: A * q = -B * Q (1)
  • add corrections to the Fock matrix due to these charges:

Fnew = F + BT * q (2)

Linear-scaling COSMO implementation uses the conjugated gradients method [1] to solve the system of equations (1). This iterative method allow to solve the systems of linear equations without storing matrices A and B, thus avoiding excessive memory requirements of the Cholessky method used in conventional COSMO. The matrix vector products A * q, B * Q and BT * q are computed using modified version of the adaptive fast multipole method [2]. This method uses O(N) operations to compute the products and doesn't require evaluations of the matrices A and B. All other calculations are performed similarly to the conventional COSMO.

 

1. R. Barret, M. Berry, T.F. Chan, J. Demmel, J. Donato, J. Dongarra, V. Eijkhout, R. Pozo, C. Romine, and H. van der Vorst "Templates for the solution of linear systems: building blocks for iterative methods", SIAM, Philadelphia, 1994.

2. H. Cheng, L. Greengard and V. Rokhlin " A fast adaptive multipole algorithm in three dimensions", J. Comp. Phys., 1999, 155, 468-498.
