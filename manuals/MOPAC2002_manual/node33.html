New Parameters

As new parameter sets become available, a mechanism must exist to allow these parameters to be used by MOPAC. The simplest way is to convert the parameters into a small data set, and then to make this data set known to MOPAC, by use of the keyword EXTERNAL. Only the unique parameters need be defined in the data set, any derived parameters are computed internally.
