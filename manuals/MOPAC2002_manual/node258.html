PDBOUT

To be used only with proteins, PDBOUT will cause the geometry to be printed in PDB format. Although the format printed is designed to match that used in the Brookhaven Protein Data Bank, the agreement is not exact. The main differences are:
  • Only the geometry is output. No preamble is generated.
  • The terminus of the protein is not identified.
  • Every atom in a residue is assumed to belong to the residue. If a group is attached to a residue, it is considered to be part of the residue.
  • The numbering system(s) used in the files deposited in the PDB are not consistent. Therefore the convention defined by the PDB itself is used.
