SCF-CI

The C.I. in MOPAC is sufficiently versatile to allow almost any closed or open shell system, in any state, to be calculated. Unpaired spin densities can be generated. Geometry optimization can be performed on both ground and excited states, including degenerate states (both with and without Jahn Teller effects). This can be contrasted to the MCSCF method, which is not supported, in which the SCF is performed using several configurations.
