MOZYME

The keyword MOZYME alters the way in which the SCF is generated. When MOZYME is used, calculations on large systems require only a small fraction (a few percent) of the memory needed for an equivalent conventional calculation, and run very much faster (for 4,000 atoms, the jobs run about 200 times faster.)
