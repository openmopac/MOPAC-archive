CCL4

Because the extra data for the MST calculations is so complicated, the keyword CCL4 has been written. When CCL4 is present, the extra data required by TOM is supplied by MOPAC. While this makes MST calculations easier to run, it does reduce flexibility. When CCL4 is present, the following conditions are used:

  • The solvent is assumed to be carbon tetrachloride.
  • The dielectric is set to 2.228 (the value for CCl4).
  • The temperature is set to 298.15K.
  • The volume of the solvent molecule is set to 97.07Å3.
  • The diameter of the solvent molecule is set to 5.38Å.
  • The thermal expansion coefficient is set to 0.00124K-1.
  • The surface tension is set to 0.0 dyne-cm-2.
  • The surface tension derivative is set to 0.0.
  • The cavity microscopic coefficient is set to 0.0.
  • The ORT option is used.
  • The Van der Waals radii as defined in Orozco, Bachs, and Luque, J. Comp. Chem., 16 563-575 (1995) are used.
