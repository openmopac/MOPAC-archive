CROSS

When CROSS is specified, the minimum energy geometry subject to the constraint that two electronic states are degenerate is calculated using the method of Anglada and Bofill [28]. In other words, the lowest energy intersystem crossing geometry is generated.

For CROSS to work, an electronic excited state must be defined using ROOT=n, in which case the states n and n-1 will be used in the intersystem crossing. These states can be further defined by use of keywords such as SINGLETor TRIPLET, etc. For example, for a singlet-triplet intersystem crossing, ROOT=2 could be used. For a singlet-singlet intersystem crossing ROOT=2 SINGLET might be suitable, and for a triplet-triplet crossing, ROOT=2 TRIPLET could be used.

CROSS works best when the states start off degenerate, or almost degenerate. If they are far apart in energy, the calculation will likely fail.

In addition to CROSS and ROOT=n, other keywords, defining the C.I., will be needed.

A recommended sequence of  steps involved in a CROSS calculation are:

(A) Optimize the geometry of the high-energy system (the reactant).
(B) Re-optimize the geometry using C.I. of the size needed in the CROSS calculation. 

Somehow generate a geometry where the desired states are degenerate.  This requires a good knowledge of the system involved.  For example:

(C) Set up a reaction path to move in the direction of the low energy form.
(D) Identify the geometry where the desired states are degenerate.
(E) Run the CROSS calculation.

Example of a CROSS calculation

Consider the decomposition of 1,2-dioxetane into two molecules of formaldehyde, in which the crossing point wanted involves a singlet and a triplet state having the same energy.  This is a little artificial, but illustrates the steps involved.

First, set up a guessed geometry for 1,2-dioxetane, Figure 1.  Two dummy atoms are used in this geometry; they are present so that later on a reaction path can be set up, in which the two oxygen atoms are forced apart.  The first dummy atom is defined as being midway between the two carbon atoms, and the second dummy atom is defined as being midway between the two oxygen atoms.  This is achieved using SYMMETRY and the (locked) angles of 180 degrees. The system has C2v symmetry.

After optimizing the geometry, a reaction path is set up, Figure 2. In this, the O-O distance starts at 2*0.656 Angstroms, and increases to 1.4, 1.6, 1.8, etc. A.  The geometry is allowed to optimize, subject to the symmetry constraint that it remains C2v. During this reaction, the triplet excited state drops in energy, until at an O-O distance of 2.0 Angstroms it became very small.

GEO-OK is used to prevent the run failing when degenerate levels are encountered - that is expected in this type of calculation.

C.I.=2 is used because a C.I. is necessary.  In more realistic calculations, higher C.I.s might be needed, e.g., C.I.=4.

MECI is used, as we want to monitor the singlet-triplet gap.

Keywords STEP=n.nn and POINT=n are not used, as we need full outputs for each optimized geometry.

At an O-O separation of 2 A, the singlet-triplet energy difference is small.  Normally, the optimized geometry at that point would be used to start the CROSS calculation, but because this system is very simple, the original geometry from Figure 2 was used, with the O-O separation set to 2 A.

A common problem with systems that have symmetry is that the symmetry is retained, even when it should be lost.  To avoid this, the angle and dihedral of one oxygen are perturbed, by 5 degrees.  In addition, the symmetry constraints on the hydrogen atoms are removed.  The only symmetry retained is that the dummy atoms should be exactly half way between the carbon atoms and between the oxygen atoms. In other words, all symmetry is removed, and the system perturbed to remove the original symmetry.
