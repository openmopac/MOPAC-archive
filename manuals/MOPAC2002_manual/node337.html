VDW(text)

The Van der Waals radii of atoms can be set or changed by use of VDW. The format of the command is:

VDW(:<chemical symbol1>=n.nn:<chemical symbol2>=n.nn:<chemical symbol3>=n.nn:...)

For example, VDW(:Cl=2.33:Br=2.50) would override the default values of the van der Waals radii of chlorine and bromine (1.65 and 1.80, respectively). Note that all chemical symbols (including the first one) must be preceded by a colon (:).

This keyword is used in setting the VDW radii in the COSMO solvation method.

The VDW radii are also used in determining connectivity in the generation of the Lewis structure. Thus, to make sure that a magnesium atom is fully ionic, the keyword VDW(:Mg=-1.0) could be used. Note that a large negative VDW radius ensures that no atoms are within the VDW radius.
