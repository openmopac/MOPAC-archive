PDB input/output

Molecular geometries can be read in using Protein Data Bank format. Regardless of the input format, geometries can be output in PDB format, under user control. There are several commonly used minor variations on the PDB format definition, most of which are accommodated in the input, and standard PDB format is used in the output.
