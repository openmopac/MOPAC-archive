LEWIS

In MOZYME, this prints out the connectivity table generated by subroutine LEWIS, and the Lewis structure generated by MAKVEC.

Because LEWIS is intended to be used as a diagnostic tool only, the calculation is stopped after the Lewis structure is printed. An SCF will not be run.
