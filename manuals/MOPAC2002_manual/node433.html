Methane

In this calculation, methane is specified, and an OPEN(6,3) is used. Normally, OPEN(n1,n2) implies a C.I. calculation, but since the result is obvious (A C.I. on a simple closed-shell system involving only one microstate does exactly nothing.), C.I. is not used.
