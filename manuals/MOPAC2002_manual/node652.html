Structure of the Tables

Because of the size of these Tables, finding any specific system can be quite time-consuming. To help locate species, the order of occurrence of species within a Table is determined by the following rule:

  • The position in a table of a given compound is determined by its empirical formula. Each element is given a weight depending on its atomic number, with lighter elements occurring before heavier elements, and electrons being given a very small weight. For example, H2 < CH4 < C10H22 < CH2O < Na(+) < Na < Na(-).

Elements in the empirical formula are arranged as in J. O. Cox, G. Pilcher, "Thermochemistry of Organic and Organometallic Compounds", Academic Press, New York, N.Y., 1970.
