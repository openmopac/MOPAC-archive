Fortran Files

DENSITY consists of several FORTRAN-77 files. These are:

CNTOUR A general 2 or 3-dimensional contour-drawing subroutine.
DRAW A subroutine to draw one contour.
EULER Rotates a point in 3-D space.
DATIN Reads in data defining one particular picture.
MAIN Main segment; generates array for contouring.
MAXMIN Calculates highest and lowest points in the picture.
PLOTGR Interface between FORTRAN-77 and graphics routine.
READIN Gets in data defining molecule
PLOTGR Generates data for graph plotter.
