NODIRECT

Do not use the default direct SCF method in MOZYME.  By default, all two-electron two-center integrals are calculated as needed.  If NODIRECT is used, then these integrals would be calculated once per SCF.  NODIRECT should not be used for routine work.  See also SEMIDIRECT.
