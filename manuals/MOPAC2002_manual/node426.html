Running DENSITY

The command density followed by a filename will start an on-line DENSITY job. E.g. density ethylene, where ethylene.gra is the data-set for DENSITY, and a calculation of ethylene had been run earlier using MOPAC.

The main files that are required by DENSITY are:

<filename>.gra
