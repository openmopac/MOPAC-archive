COMPFG

When COMPFG is specified, the DHf  is printed whenever COMPFG is called. In a conventional calculation (i.e., MOZ is not used), if DEBUG is also specified then the first few coordinates are printed, and if LARGE and DEBUG are both specified, then the whole geometry is printed.
