Keywords

Two sets of key-words are provided. At the start of a run the user must provide information about the source of the molecular data. At present, this data can come from one of two sources. These are (a) off the data-file itself, which is the MANUAL mode, and (b) from a disk or restart-type file, which was created by an earlier run using MOPAC.
