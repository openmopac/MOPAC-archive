meci.h

The maximum values of the variables used in a Multi-Electron Configuration Interaction calculation are stored in meci.h. These variables are described in the file. Normally, the values supplied are sufficient for most work. However, if you want, NMECI can be changed. Setting it to 1 will save some space, but will prevent all C.I. calculations except simple radicals. A useful value for NMECI is 4. This will allow simple excited states to be run.
