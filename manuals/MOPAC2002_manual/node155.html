DEP

For use only with EXTERNAL=<filename>. When new parameters are published, they can be entered at run-time by using EXTERNAL=, but as this is somewhat clumsy, a permanent change can be made by use of DEP.

If DEP is invoked, a complete FORTRAN module will be generated, and this can be inserted directly into the MOPAC source code directory.

Note that the name of the module generated will correspond to the method specified by the keywords.
