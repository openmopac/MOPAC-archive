CHCL3

Because the extra data for the MST calculations is so complicated, the keyword CHCL3 has been written. When CHCL3 is present, the extra data required by TOM is supplied by MOPAC. While this makes MST calculations easier to run, it does reduce flexibility. When CHCL3 is present, the following conditions are used:

  • The solvent is assumed to be chloroform.
  • The dielectric is set to 4.717 (the value for chloroform).
  • The temperature is set to 298.15K.
  • The volume of the solvent molecule is set to 80.66Å3.
  • The diameter of the solvent molecule is set to 5.02Å.
  • The thermal expansion coefficient is set to 0.00127K-1.
  • The surface tension is set to 0.0 dyne-cm-2.
  • The surface tension derivative is set to 0.0.
  • The cavity microscopic coefficient is set to 0.0.
  • The ORT option is used.
  • The Van der Waals radii as defined in Orozco, Bachs, and Luque, J. Comp. Chem., 16 563-575 (1995) are used.
