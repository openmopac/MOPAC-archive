NSPA=n

Used for the COSMO method (see EPS) to set the number of geometrical segments per molecule to n. This keyword should seldom be used, as the default value (42) should be adequate. However, for high-precision work a larger value from the set 3i x j2 x 10 + 2  should be used (e.g., 92,122, or 162).  See also DVFILL
