H2O

Because the extra data for the MST calculations is so complicated, the keyword H2O has been written. When H2O is present, the extra data required by TOM is supplied by MOPAC. While this makes MST calculations easier to run, it does reduce flexibility. When H2O is present, the following conditions are used:

  • The solvent is assumed to be water.
  • The dielectric is set to 78.5 (the value for water).
  • The temperature is set to 298.15K.
  • The volume of the solvent molecule is set to 18.07Å3.
  • The diameter of the solvent molecule is set to 2.77Å.
  • The thermal expansion coefficient is set to 0.00026K-1.
  • The surface tension is set to 71.69 dyne-cm-2.
  • The surface tension derivative is set to 0.657.
  • The cavity microscopic coefficient is set to 1.227.
  • The ORT option is used.
  • The Van der Waals radii as defined in Orozco, Bachs, and Luque, J. Comp. Chem., 16 563-575 (1995) are used.

These conditions allow the results reported in J. Comp. Chem, Vol 16, p. 586, Table II, option to be reproduced. From Orozco, et. al.'s article, the preferred method is AM1.
