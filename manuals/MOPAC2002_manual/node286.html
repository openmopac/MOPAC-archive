RESIDUES

RESIDUES will put a label on each atom in a protein. The label is a short description of the atom, and is useful in identifying the atoms in a protein. The general form of the label for an atom is '(nnnnn TYPfrrr)' where nnnnn is the atom number (the 10th atom would have the atom number 10), TYP is a three letter symbol for each amino acid (the allowed symbols are shown in the Table). If a residue is not recognized, then the symbol 'UNK' (unknown) will be used. Backbone atoms are indicated by an asterisk (*) after the residue symbol, i.e., `f'=*; for all other atoms, 'f' is a space; rrr is the residue number. A modified residue can still be recognized if ZENO=text is used. The residue nearest to the NH2 end of the protein is No. 1, the next is 2, and so on.

Small molecules are often found associated with proteins. To allow for this, some special groups are also recognized. A phosphate group is indicated by the symbol 'PO4'. In place of the residue number, the number of the phosphate will be given, the first phosphate being 1. More special groups will be added as needed. Hydrogens added to these small molecules are given the symbol of the chemical element to which they are attached. If RESEQ is used, then RESIDUES is automatically run.
