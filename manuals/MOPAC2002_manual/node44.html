Graphics

Although MOPAC does not have any graphics, it has the capability of generating data for use by graphics packages. This data includes information on the molecular orbitals, the overlap matrix, and the orbital exponents, etc. Graphics packages can then use the data to generate contour maps of M.O.s, density, density differences, etc.
