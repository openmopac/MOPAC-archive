OPTBACK

Only to be used with MOZYME. In a protein, the optimization flags for all backbone atoms are set to ' 1 '. The optimization flags for all atoms attached to the backbone (that is, the side-chains and the hydrogens and oxygens of the -NH-CHR-CO-) are set to ' 0 '.
