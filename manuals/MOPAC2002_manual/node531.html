Other Considerations

The first time a large system is run, there is a high probability that errors will exist in the data-set which would make the full calculation useless. To save time, a check is run on the system to determine the number of each type of LMO. If an atom bonds to more than four neighbors, then the s framework cannot be made, and the run is stopped with a warning message.

If the calculated net charge is different from the charge specified on the keyword line, then the assigned number of occupied LMOs will not match the calculated number of occupied LMOs, i.e., the number of s and p LMOS plus the lone pairs plus the anions. If that happens, then the job is stopped, and a warning message printed out.
