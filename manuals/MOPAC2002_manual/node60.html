Determining Net Charge

Determining the net change on a large system, such as a protein, can be tedious. The simplest way would involve examination of the structure, and identifying each ionized site. This task can be performed rapidly by running a simple calculation. If a MOZYME calculation is run, then any ionized sites, and the net charge on the system, will be printed. It is not necessary to perform an SCF calculation, so this operation is very rapid.

Frequently, the geometry supplied contains errors. If any of these errors would prevent a meaningful calculation being run, then a brief description of the errors detected is printed. This speeds up the preparation of data sets.
