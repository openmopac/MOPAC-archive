DENSITY

DENSITY, which comes with MOPAC, is an electron-density plotting program. It accepts data-files directly from MOPAC, and is intended to be used for   graphical representation of electron density distribution, individual M.O.s, and difference maps.

Although DENSITY is now quite dated--there are better GUI programs readily available--it is still useful as a simple program for stand-alone work.
