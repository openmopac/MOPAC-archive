Example of Data: Ethylene

For this example we will use the p system in ethylene. Two pictures are to be drawn based on the results from an earlier MOPAC calculation. The first plot is in the plane parallel to the plane of the molecule, the second one is in the plane perpendicular to the C-C axis. DENSITY data-files should have the suffix .gra.

  

Figure: DENSITY data-set for Ethylene

Line 1 :  DEBUG
Line 2 :  Ethylene pi system
Line 3a:  CENTER=(0.7,0.0,0.5) LINE=(0.0,0.0,1.0) EDGE=2.0 HOMO
Line 3b:  CENTER=(0.7,0.0,0.5) LINE=2 EDGE=2.0 HOMO
Line 4 :

Line 1: First set of key-words. These control the mode of operation.
Line 2: Title of plot. Up to 79 characters can be used.
Line 3a: Second set of key-words. A full list of key-words, and their definitions, is given in the next Section.
Line 3b: Key-words for the second picture.
Line 4: End the input with a blank line.
