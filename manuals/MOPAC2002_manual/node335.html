UHF

The unrestricted Hartree-Fock Hamiltonian is to be used.

For open shell systems involving many atoms, geometry optimization normally runs faster when UHF is used.  This is because the open-shell RHF gradient calculation is relatively slow.

A limitation of the UHF method is that the resulting wavefunctions are not spin quantized, that is, although <Sz> has a  well-defined value, e.g. 0, 1/2, 1, 3/3, etc., <S2> is normally not quantized.  For a system with an <Sz> of 1/2, the <S2> should be 3/4 = 1/2*(1/2+1), but is usually higher.  This is a consequence of spin contamination.  Spin contamination is not present if UHF is not used.
