ADD

When two or more pictures are to be added together to form a single picture, ADD must be used. For example, a user might want to add or subtract two M.O.s to see the effect. Thus if M.O. 2 and M.O. 3 were to be added together, the data for such a calculation could be: 
Line 1:
Line 2:   FORMALDEHYDE
Line 3: CENTER=2 LINE=(0,0,1) EDGE=2 PSI=2 ADD
Line 4: CENTER=2 LINE=(0,0,1) EDGE=2 PSI=3 
Line 5:
See also PHASE.
