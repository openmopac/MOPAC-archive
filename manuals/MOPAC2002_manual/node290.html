RMEM=n.nn

The amount of real dynamic memory assigned to a calculation can be changed by RMEM=nn. If nn is positive, the amount of memory, in units of array elements, is increased by nn, decreased if nn is negative. This keyword should be used in the event that the program runs out of real memory before the calculation is complete.
