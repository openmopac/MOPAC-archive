SEMIDIRECT

Do not use the default direct SCF method in MOZYME.  By default, most two-electron two-center integrals are calculated as needed.  If SEMIDIRECT is used, then the NDDO set of integrals (100 between any two atoms with an s-p basis set that are near to each other) are calculated once per SCF.  All other integrals (point charge - point charge and dipole - point charge) are calculated as needed..  SEMIDIRECT should not be used for routine work. See also NODIRECT
