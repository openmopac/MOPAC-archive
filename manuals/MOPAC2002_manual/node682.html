TOLERX "Test on X Satisfied"

 
Defined in:
FLEPO
Default value:
0.0001 Ångstroms
Basic test:
The projected change in geometry is less than TOLERX Ångstroms.
Exception:
If GNORM is specified, the TOLERX test is not used.
