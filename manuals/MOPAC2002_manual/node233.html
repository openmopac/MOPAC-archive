NLMO=nnn

The average number of atoms in a localized molecular orbital is defined as nnn. The default is usually adequate, if it is not then an advisory message will be printed, and the number can be raised using NLMO=nnn. The amount of memory needed can be lowered by reducing nnn. However, lowering it too much will result in the job failing. If that happens, follow the instructions in the output in order to correct the fault.
