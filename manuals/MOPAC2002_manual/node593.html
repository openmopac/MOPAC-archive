Basic Physical Constants

All basic physical constants should be taken from: "Quantities, Units and Symbols in Physical Chemistry," Blackwell Scientific Publications Ltd, Oxford OX2 0EL, UK, 1987 (IUPAC, based on CODATA of ICSU, 1986 [71]). pp. 81-82. A summary of these constants is given below..

Some derived quantities, which will be used in this section only are: Moment of inertia: I 1 amu Ångstrom2 = 1.660540x10-40g cm2.

Rotational constants: A, B, and C (e.g. ) A = h/(8p2I)
With I in amu Ångstroms2 then: A (in MHz) = 5.053791x105/I 
A (in cm-1) =  5.053791x105/cI  = 16.85763/I
