Use of the SADDLE technique to locate a transition state

  • Optimize the geometry of the reactant(s).
  • Using the same atoms in the same sequence, optimize the geometry of the products.
  • To save time, run a LOCATE-TS calculation to move the reactants and products nearer to each other.  A typical example would be LOCATE-TS=(C:10,10;)
  • Run a SADDLE calculation using either the reactant(s) and product(s) or the "first" and "second" geometries from the LOCATE-TS job.
    For neatness, format the data-set as follows:
    GEO_DAT="text" +
GEO_REF="text" +
SADDLE plus other keywords
  • Most calculations end because "both reactants and products are on the same side of the transition state" If this happens, examine the output and find a geometry with a large COSINE.  Anything over about 0.7 is good.  The smaller the value of "DISTANCE A - B" the better, anything below about 0.2 should be okay.  Use that geometry to start a transition state refinement calculation and IRC run.
