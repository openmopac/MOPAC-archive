How to set up a data-set for a Solid-State Calculation

 

For examples of data-sets for solids, please see: Periodic Table of Solids

A solid-state data set can be identified by the presence of three "Tv" atoms at the end of the geometry. 

The rewritten program MAKPOL can easily construct solid-state data-sets for use with MOPAC.  because of this, further explanation here is not useful.
