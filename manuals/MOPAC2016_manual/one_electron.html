The matrix elements between orbitals φλand φσ on the same atom are calculated from the electron-nuclear attraction energy, and also from the Uλλ value if λ = σ.

The one-electron matrix is unaffected by (a) the charge and (b) the electron density. It is only a function of the geometry. Abbreviation: 1ELEC.
