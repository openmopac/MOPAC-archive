Getting started using PARAM

Download and Unpacking

Download the file http://openmopac.net/PARAM.zip Move the file PARAM.zip to where you want PARAM to be.  PARAM must be on a Windows machine, an appropriate location would be "C:\Users\yourname" another would be any local drive that would be suitable, for example if you want drive M:\ to be used, move PARAM.zip to M:\   Unpack PARAM.zip using the appropriate password.

Setting up the PARAM folder

All the instructions for preparing a full working copy of PARAM are given in the file README.txt in the top folder of PARAM.
